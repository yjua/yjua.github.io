常用的subliemtext 插件

sublime text

package control

sublime 的包管理工具，安装方式从官网
根据选择安装

Emmet

是一个可以让我们更快速更高效编写HTML和CSS的插件。在package中搜索Emmet安装。

HTML-CSS-JS Prettify

HTML, CSS, JavaScript and JSON code formatter for Sublime Text 2 and 3 via node.js

Minify

minify是一个可以快速压缩html、css、js、svg文件的插件

Seti_ui

这是一个sublime text3的theme。很不错。

Side​Bar​Enhancements

这个是一个sublime右键增强的插件，安装之后，很多功能可以通过鼠标右键直接操作。
