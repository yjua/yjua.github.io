JavaScript中this常见的一些用法

JavaScript

JavaScript中this是个很重要，但是又很容易混乱的一个东西，尤其是在各种异步操作的时候，一不小心就很容易出错。

this是JavaScript中一个关键字。它指的是在运行时状态下所指的一个对象。

下面是一些常见的情况。

全局上下文

在全局上下文中，this代指的是全局对象。

函数上下文

在函数内部调用，this主要取决于当时的环境

这是最常用的一种调用方法，因为fn1函数实在全局对象下直接调用，所以fn1()和window.fn1()是等价的。这个时候this其实上是指向调用当前方法的对象。

我们直接调用fn2 相当于window对象调用，则this指向了全局对象，所以结果为10。当执行obj.fn2()的时候，this指向了当前调用对象obj,则结果为20；

构造函数中的this

在构造函数中的this一般指向通过该构造函数实例化后的对象。

apply 和 call

这两个方法可以让函数在调用的时候指向一个指定的对象。

bind函数

当一个函数fn调用bind函数的时候，如fn.bind(obj)会创建一个跟fn一样的函数体。但是创建的这个新的函数体的this会指向调用bind时候的第一个参数。
