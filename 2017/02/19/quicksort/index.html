JavaScript常用算法-快速排序

JavaScript 算法

快速排序是几种最常见的排序算法之一。使用较广、速度较快。

快速排序的基本思路可以分成三步:

  • 在要排序的数组中找一个基准。
  • 把所有比它小的放到一侧，把所有比他大的放到另一侧。
  • 再把两侧的分别当做一个独立的整体，递归进行快速排序，直到最后排序完毕。
