JavaScript常用算法-插入排序

JavaScript 算法

插入排序是一种直观简单的排序算法。简单来讲，就是把未排序的数据，跟已排序的数据进行比较,然后插入合适的位置。

算法实现步骤（从小到大排序）：

  • 从第一个元素开始，一般认为第一个元素已经排序。
  • 取出下一个要排序的新元素，从已经排序的最后一个数据向前进行比较。
  • 如果该元素（已排序）大于新元素，则将该元素向后移动一个位置。
  • 重复第三步，直到找到已排序的元素小于或等于新元素的位置
  • 插入该新元素
  • 重复2-6步骤

算法复杂度:
最好的情况是：o(n)
最坏的情况是：o(n^2)
平均复杂度是：o(n^2)

插入排序适合数量比较少的数组排序，(比如数量不过千)。
